# SeenSnap Phase 1 Milestones

## Milestone 1: Foundation

- Monorepo created
- FastAPI and Expo shells created
- PostgreSQL schema and Alembic baseline added
- Environment templates documented

## Milestone 2: Auth and profile

- Apple login
- Google login
- user bootstrap
- profile setup and preferences

## Milestone 3: Content core

- TMDB search
- title detail
- watchlist
- streaming option ordering

## Milestone 4: Snip flow

- image upload to GCS
- assisted image match pipeline
- low-confidence fallback to manual title confirmation
- result and retry states

## Milestone 5: Ratings, reviews, and recommendations

- ratings and optional review flow
- home recommendations
- team-aware ranking inputs

## Milestone 6: Teams and sharing

- team create and join by link/code
- team activity feed
- Instagram share handoff

## Milestone 7: Launch hardening

- push notifications
- affiliate click tracking
- observability
- App Store and Play Store submission readiness
